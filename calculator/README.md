# Calculator Program

This program is a simple command-line calculator that evaluates mathematical expressions.

## Usage

To use the calculator, run the `main.py` script with your expression as an argument.

```bash
python main.py "<expression>"
```

Replace `<expression>` with the mathematical expression you want to evaluate. Make sure to enclose the expression in double quotes if it contains spaces or special characters that your shell might interpret.

## Examples

* **Addition:**
  ```bash
  python main.py "3 + 5"
  ```

* **Subtraction:**
  ```bash
  python main.py "10 - 4"
  ```

* **Multiplication:**
  ```bash
  python main.py "6 * 7"
  ```

* **Division:**
  ```bash
  python main.py "20 / 5"
  ```

* **Mixed operations:**
  ```bash
  python main.py "(3 + 5) * 2"
  ```

## Error Handling

The program will display an error message if the expression is invalid or empty.

```bash
Error: Invalid expression
```
